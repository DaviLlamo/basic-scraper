# Scraper basico
### Uma pequena advertencia, esse projeto não busca de forma nenhuma ferir a imagem do mercado livre ou expor alguma fraqueza do site em relação a segurança ou informações, esse pequeno scraper é basico, sem objetivos de má indole com a plataforma em relação, apenas com o intuito de meu aprendizado e upar aqui no github como forma de portifolio dito isso espero que aproveite esse repositorio
Um simples projeto de scraper do site de vendas do Mercado Livre, código feito com python, coletando o informações dos produto ( preço, nome e link).
o usuario deve executar  o arquivo exe e colocar a url do mercado livre desejada, o scraper abre o url e pega as informações e as organiza em um arquivo excel (.xlsx) baixado na maquina do usuario.

## Funcionalidades
- Coletar nome de produtos
- Coletar preço dos produtos
- Extrair links dos produtos
- Exportar os dados para Excel
- Interface simples para inserir a URL

## Tecnologias utilizadas
- Python
- Requests
- BeautifulSoup
- Pandas
- Tkinter
- openpyxl


## Objetivo do projeto
Este projeto foi criado como um exemplo básico de automação e coleta de dados da web (web scraping), demonstrando como extrair informações de páginas da internet e organizá-las em arquivos estruturados.


## como rodar

Instale o arquivo txt no terminal:

pip install -r requirements.txt

rode o arquivo py colocado no projeto

tambem deixarei um link com o app em arquivo .exe, um executavel apenas clicar e verá o app rodar direto
